# 介紹各個應用程式
使用者可以使用Replit如何使用Codespace、Gitpod

# 如何使用Codespaces維護自己倉儲?
可使用init_replit指令安裝Python模組
chmod u+x init_replit

./init_replit

接著輸入(這是動態網站)

python3 main.py

再開啟新的終端機

python3 -m http.server

這是靜態網站

即可編輯網站
