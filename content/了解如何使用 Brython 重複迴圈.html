了解如何使用 Brython 重複迴圈

Brython 提供了兩種常用的重複迴圈：for 迴圈和 while 迴圈。

1. for 迴圈

for 迴圈用於迭代可迭代對象中的元素。可迭代對象可以是列表、字符串、元組或字典等。

以下是 for 迴圈的語法：

for variable in iterable:
  # 迴圈體

其中：

  • variable 是用於遍歷可迭代對象的變量。
  • iterable 是要遍歷的可迭代對象。

以下是 for 迴圈的示例：

<!DOCTYPE html>
<html>
<head>
  <title>Brython Example</title>
  <script src="https://cdn.jsdelivr.net/npm/brython@3.10.0/dist/brython.js"></script>
</head>
<body>
  <script type="text/python">
    numbers = [1, 2, 3, 4, 5]

    for number in numbers:
      print(number)
  </script>
</body>
</html>

此代碼將打印以下輸出：

1

2

3

4

5

2. while 迴圈

while 迴圈用於重複執行程式碼塊，直到指定的條件為假。

以下是 while 迴圈的語法：

while condition:
  # 迴圈體

其中：

  • condition 是要檢查的條件。

以下是 while 迴圈的示例：

<!DOCTYPE html>
<html>
<head>
  <title>Brython Example</title>
  <script src="https://cdn.jsdelivr.net/npm/brython@3.10.0/dist/brython.js"></script>
</head>
<body>
  <script type="text/python">
    i = 0

    while i < 5:
      print(i)
      i += 1
  </script>
</body>
</html>

此代碼將打印以下輸出：

0
1
2
3
4
